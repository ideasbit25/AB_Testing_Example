# A/B Testing Introduction

A simple Python example demonstrating the basics of A/B testing using statistical hypothesis testing.

## Overview

This project compares the conversion rates of two webpage variations (Control and Experiment) to determine whether the observed difference is statistically significant.

In this example:

- Control Group: Blue Button
- Experimental Group: Red Button
- Metric: Conversion Rate
- Statistical Test: Two-sample t-test (using SciPy)

## Requirements

- Python 3.x
- NumPy
- SciPy

## Example Output

```

```

## Concepts Covered

- A/B Testing
